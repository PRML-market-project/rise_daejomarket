# rise_daejomarket

## Run all services

From the repository root:

```powershell
npm.cmd run dev
```

or:

```powershell
powershell -ExecutionPolicy Bypass -File .\run-all.ps1
```

This starts:

- backend: http://localhost:8080
- ai-server: http://localhost:8000
- frontend: http://localhost:5173
- admin-frontend: http://localhost:3000
